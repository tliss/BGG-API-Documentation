Understanding XML

Looking at the XML

This will be pretty long, but lets take a look at the XML from our previous request:

http://www.boardgamegeek.com/xmlapi/boardgame/13
Open XML

This XML contains all BGG's database information for the boardgame with the ID number 13. Just taking a quick glance through it, you can probably identify some common tags. The name tags all identify alternative names for the game, while the name tag with the primary="true" attribute is the game's primary name.
